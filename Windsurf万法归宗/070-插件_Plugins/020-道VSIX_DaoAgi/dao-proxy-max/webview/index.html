道·外接api

道生一 · 一生二 · 二生三 · 三生万物

第三方 API 模型无感接入 Windsurf / VSCode. 本面板把配置 / 探针 / 注入 / 诊断 / 日志 汇归一处.

心法三则

  • 不抢道 — 添加模型入口, 绝不覆盖官方条目. 所有 vendor/模型以 dao-* 前缀出现.
  • 最小本源 — 零依赖 (仅 Node 内置 http/https). 新增 provider = 配置改 + 零代码改.
  • 表里相一 — Anthropic 入 → Anthropic 出; OpenAI 入 → OpenAI 出. 协议透明.

支持的 Provider (14 家)

  • Anthropic-native · Anthropic-Compat (任 OneAPI/NewAPI/自建)
  • OpenAI · OpenAI-Compat
  • DeepSeek · Moonshot(Kimi) · Qwen · GLM
  • Gemini · OpenRouter · SiliconFlow · Groq · Mistral
  • Ollama · LM Studio · vLLM · GitHub Models (免费)
