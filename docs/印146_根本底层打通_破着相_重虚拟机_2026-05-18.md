### § 4.3 · 中间一坑 (帛书六十四「为之于其未有也」)

VM 内 dao_proxy 启时 **pool=0**:
- vm_proxy_deploy.js L364-370 上传 `tokens_dao_123.txt` + `tokens_ws_59.txt`
- 但**主公本机这两文件不存** (硬编码名 · 印 123 时刻)
- → omniPutFile 跳 (L417: `if (!fs.existsSync) continue`) → VM 之 tokens_dao.txt 不存 → dao_proxy 之 4 源 (env DEVIN_TOKEN / env DEVIN_TOKENS / DAO_TOKENS_FILE / wam-state.json) 全空 → pool 空

**治** (印 146 加 · 帛书廿八「朴散则为器」):
```powershell
# 1. 抽 wam-state.json:activeApiKey
$wam = Get-Content "$env:USERPROFILE\.wam\wam-state.json" -Raw | ConvertFrom-Json
$tok = $wam.activeApiKey   # devin-session-token$JWT

# 2. 写本机 BASE_DIR/tokens_dao_123.txt (供下次 deploy 复用)
$tok | Out-File "$BASE_DIR\tokens_dao_123.txt" -Encoding ascii -NoNewline

# 3. 当下: omni /_/file/ PUT 上传 + /_/run 重启 dao_proxy
$body = $tok + "`n"
Invoke-RestMethod "$vmBase/_/file/home/ubuntu/dao_proxy/tokens_dao.txt" -Method PUT \
  -ContentType 'application/octet-stream' -Headers @{ Authorization = $basicAuth } -Body $body

Invoke-RestMethod "$vmBase/_/run" -Method POST -ContentType 'application/json' \
  -Headers @{ Authorization = $basicAuth } \
  -Body '{"cmd":"cd /home/ubuntu/dao_proxy && bash start.sh","timeout":15000}'
```

实证: VM 内 dao_proxy 重启后 **pool.total=1** · chat 真转。
