### 问题 ② · METRICS 计数不自洽 (无 inflight)

真测后 `/metrics` 显:

```json

```

**8 ≠ 0 + 2 = 2** — 余 6 笔在 client timeout 后**后台仍跑**, METRICS 无字段反此. 主公看不清池态.

**治** (帛书六十四 · 慎终若始 · 透明):

```js
const METRICS = {
  ...
  inflight: { total: 0 },     // 印 140 加
  ...
};
function recReq(t) {
  METRICS.requests.total++;
  METRICS.inflight.total++;   // 加 1
  ...
}
function recOk(ms) {
  METRICS.successes.total++;
  METRICS.inflight.total = Math.max(0, METRICS.inflight.total - 1);   // 减 1
  ...
}
function recErr() {
  METRICS.errors.total++;
  METRICS.inflight.total = Math.max(0, METRICS.inflight.total - 1);   // 减 1
}
```

